## Описание

Плагин VIP_WeaponPack позволяет VIP-игрокам получать готовые комплекты оружия в начале раунда. Это улучшенная версия с исправлениями ошибок и новыми функциями для лучшего игрового опыта.

## Функции

✅ **Автоматическое меню в начале раунда** - VIP-игроки получают меню выбора оружия  
✅ **Поддержка mp_halftime** - Корректная работа при смене сторон команд  
✅ **Предотвращение дублирования** - Игрок не может получить оружие дважды за раунд  
✅ **Система кулдауна** - Ограничение на повторное использование по раундам  
✅ **Опция показа меню** - Выбор: всегда, после смерти, или никогда  
✅ **Многоязычность** - Поддержка русского языка с красивыми цветами  
✅ **Правильная карта проверка** - Поддержка всех карт de_ и cs_  

## Требования к установке

- ✅ **SourceMod** - основа для плагинов
- ✅ **MetaMod Source** - загрузчик расширений
- ✅ **[VIP-Core](https://github.com/R1KO/VIP-Core)** - основной модуль VIP системы

## Установка

1. Убедитесь, что установлены **SourceMod**, **MetaMod** и **VIP-Core**

2. Скопируйте файлы в соответствующие директории:
```
addons/sourcemod/scripting/VIP_WeaponPack.sp (исходник, необязательно)
addons/sourcemod/plugins/vip/VIP_WeaponPack.smx
addons/sourcemod/translations/vip_weaponpack.phrases.txt
cfg/vip/VIP_WeaponPack.cfg
```

3. **Переводы VIP-modules** (обновите до последней версии или добавьте вручную):

Добавьте в `addons/sourcemod/translations/vip_modules.phrases.txt`:
```
"Weaponpack"
{
    "ru"		"Комплект оружий | WP"
    "en"		"Weapon Pack | WP"
    "pt"		"Pack de Armas | WP"
    "fi"		"Aseiden setti | WP"
    "ua"		"Комплект зброї | WP"
    "es"		"Set de armas | WP"
    "ar"		"مجموعة سلاح | WP"
}
```

Добавьте в `addons/sourcemod/data/vip/cfg/groups.ini`:
```
"Weaponpack"			"1"
```

5. Перезагрузите сервер

## Команды

| Команда | Описание |
|---------|---------|
| `!wp` | Открыть меню выбора оружия (если VIP активирована функция) |

## ConVar

| ConVar | Значение | Описание |
|--------|----------|----------|
| `c_Enabled` | 1 | Включить/выключить плагин (1 = включен, 0 = выключен) |
| `c_RoundLimit` | 0 | Количество раундов кулдауна (0 = без ограничений) |
| `c_RoundMenu` | 1 | Показывать ли меню в начале раунда (1 = да, 0 = нет) |

## Ключевые выборы в конфигурации

### WeaponPack.ini

Структура файла конфигурации:
```ini
"WeaponPack"
{
	"КазваниеНабора"
	{
		"Team"		"0"		// 0 - все, 2 - CT, 3 - T
		"round"		"0"		// С какого раунда доступно
		
		"weapon"	"weapon_name1"
		"weapon"	"weapon_name2"
	}
}
```


## Решение проблем

### Меню не появляется
- Проверьте, что игрок не в разминке (`c_RoundMenu` = 1)
- Убедитесь, что игрок имеет VIP статус
- Проверьте, что функция активирована в VIP системе

### Оружие не выдается
- Проверьте синтаксис `WeaponPack.ini`
- Убедитесь, что классы оружия правильные (например: `weapon_ak47`, `weapon_m4a1`)
- Проверьте консоль сервера на ошибки

### Меню появляется каждый раунд, хотя игрок выбрал "никогда"
- Это может быть если игрок выбрал "не показывать", но заходил на сервер заново
- Кук сохраняется на игроке, а не на аккаунте
